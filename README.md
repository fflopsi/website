# My personal website, written in Svelte/SvelteKit

[Click here](https://florian-frauenfelder.ch) to see the website.

## Edit it yourself

After cloning, run

```bash
pnpm install
pnpm run dev
```

Open [https://localhost:5173](https://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `src/routes/+page.svelte` and other files. The page auto-updates when you save the file.

More information about Svelte and SvelteKit can be found [here](https://svelte.dev/).
